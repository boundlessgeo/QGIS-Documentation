.. _qgisfiledownloader:

Download file
--------------
Downloads a URL on the file system. In other words you can copy/paste an URL and
download the content into a file.

Parameters
..........

``URL`` [string]
  Valid URL as a string character.

Outputs
.......

``File destination`` [file]
  Downloaded file of the chosen URL.
